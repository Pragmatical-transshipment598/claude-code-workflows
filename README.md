Use this process when you start a new task:

1. Read the task or feature request.
2. Pick the matching workflow file.
3. Copy the prompt or steps into Claude Code.
4. Ask Claude Code to help with one task at a time.
5. Review the output before you use it.
6. Run your tests or checks.
7. Update the workflow if your team finds a better order.

## 🔍 Best practices

- Keep prompts short and clear
